# 首页自定义导航栏说明

首页顶部采用微信小程序自定义导航栏方案，整体分为两行布局。  
第一行为顶部信息区：左侧公告滚动区、中间地址展示区、右侧微信胶囊安全区；第二行为独占一行的搜索框区域。

---

## 一、整体结构（2 行布局）

| 行      | 内容说明                                                   |
| ------- | ---------------------------------------------------------- |
| 第 1 行 | 左侧可滚动文案区 + 中间当前地址展示区 + 右侧微信胶囊预留区 |
| 第 2 行 | 搜索框独占一整行，支持点击跳转搜索页或聚焦搜索             |

---

## 二、各区域描述

### A. 左侧：顶部公告 / 业务状态滚动区

**功能说明**

- 用于展示动态文案：活动信息、配送时效、优惠提示、门店营业状态等。
- 文案区域支持**纵向自动轮播**（由下往上或由上往下均可，可配置）。
- 单次只展示 1 条；支持多条配置数据循环播放。
- 超出宽度时**单行省略**，不换行。
- 支持点击跳转对应活动或详情页。

**交互要求**

- 自动轮播间隔：如 2～3 秒，可配置。
- 可配置是否自动播放。
- 切换动画为纵向平滑滚动。

---

### B. 中间：收货地址展示区

**功能说明**

- 展示当前定位地址 / 当前选择收货地址。
- 地址文字长度不固定，需兼容超长文案。
- 地址区域需要预留足够**弹性空间**。
- 超长时：**单行展示** + **尾部省略号截断**，不允许换行撑高导航栏。
- 点击后跳转地址选择页。

**样式要求**

- 可包含：定位 icon + 地址文案 + 下拉箭头。
- 地址区域整体为可点击热区。
- 文案在中间区域自适应伸缩，**不能挤压右侧胶囊安全区**。

> **关键约束**：地址区域为弹性布局，需兼容超长地址文案，超出部分单行省略显示，不得影响右侧胶囊区域布局。

---

### C. 右侧：微信原生胶囊安全区

**功能说明**

- 右侧需**预留微信小程序原生胶囊按钮区域**。
- 自定义导航栏内容**不得侵入**胶囊安全区。
- 所有内容布局需**基于胶囊位置信息动态计算**。
- 保证在不同机型下与右上角胶囊保持安全间距。

**开发说明**

- 通过 `wx.getMenuButtonBoundingClientRect()`（uniapp：`uni.getMenuButtonBoundingClientRect()`）获取胶囊位置和尺寸。
- 结合状态栏高度动态计算：自定义导航栏高度、右侧留白、内容对齐方式。

> **关键约束**：导航栏采用自定义导航模式，需基于微信胶囊位置信息动态适配右侧安全区，确保文案、地址、搜索框不与原生胶囊重叠。

---

### D. 第二行：搜索功能区

**功能说明**

- 搜索框位于导航栏**第 2 行**，**独占整行**显示，不与第一行内容混排。
- 支持搜索占位文案配置。
- 点击可进入搜索页或触发搜索输入。
- 可配置右侧搜索按钮，也可仅保留搜索框。

**布局要求**

- 左右与导航栏主内容区对齐。
- 与第一行之间保留固定垂直间距。
- 高度固定，圆角样式统一。

---

## 三、适配要求（汇总）

- 导航栏高度需结合**状态栏高度**、**胶囊按钮位置**动态计算。
- 地址区采用**弹性布局**，自适应剩余空间。
- 滚动文案区和地址区需避免互相挤压。
- 所有内容需兼容：不同机型、不同状态栏高度、不同长度文案。

---

## 四、开发实现建议（技术化描述）

1. **页面配置**：使用 `navigationStyle: 'custom'`（首页已配置）。
2. **系统信息**：
   - 通过 `uni.getSystemInfoSync()` / `uni.getWindowInfo()` 获取状态栏高度。
   - 通过 `uni.getMenuButtonBoundingClientRect()` 获取胶囊位置和尺寸（微信小程序）。
3. **自定义导航栏结构拆分**：
   - `statusBar`：状态栏占位。
   - `topRow`：第一行（公告 + 地址 + 胶囊安全区）。
   - `searchRow`：第二行（搜索框）。
4. **topRow 布局**：
   - 采用 flex 布局。
   - 左侧滚动区：固定或最小宽度。
   - 中间地址区：`flex: 1`，地址文案单行溢出省略。
   - 右侧：预留胶囊安全宽度（由胶囊 left 或 windowWidth - right 计算）。
5. **公告区**：使用垂直轮播组件实现自动切换。

- 双层导航结构
- 胶囊安全区适配

## 六、一句话需求描述

首页顶部使用微信小程序自定义导航栏，分为两行：第一行为左侧公告纵向轮播、中间可点击地址区、右侧胶囊安全区；第二行为独占一行的搜索框。地址需支持超长文案单行省略，整体需基于状态栏和微信胶囊位置动态适配。
