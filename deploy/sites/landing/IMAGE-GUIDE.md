## 建议补充的素材（按优先级）

### 1. 16:9 横图（高优先级）

- **尺寸**：1920×1080 或 2560×1440
- **用途**：官网首屏 Hero 全宽背景、顶部 Banner 轮播
- **内容建议**：老年人在小区/公园上课的宽幅场景；左侧留白便于叠字
- **备注**：可从现有 1:1 图扩展画布，或重新拍摄横构图

### 2. 4:3 横图（中优先级）

- **尺寸**：1200×900
- **用途**：「云校长故事」「名师介绍」「媒体报道」模块
- **内容建议**：云校长与邻居合影、老师持证照、上课互动特写

### 3. 3:4 竖图（可选）

- **尺寸**：1080×1440
- **用途**：手机端全屏模块、抖音/视频号竖版广告
- **内容建议**：单人练习特写 + 品牌 Logo 安全区

### 4. 21:9 超宽（可选）

- **尺寸**：2560×1080
- **用途**：大屏展示、发布会背景

### 5. 透明 PNG 人物抠图（可选，利于 GSAP 动画）

- **用途**：文字与人物分层视差、入场动画
- **内容**：无背景的老师/学员半身像

### 6. 品牌固定素材

| 类型            | 尺寸               | 用途             |
| --------------- | ------------------ | ---------------- |
| Logo            | SVG 或 512×512 PNG | 导航、页脚       |
| 小程序码        | 200×200 ~ 430×430  | 页脚 CTA、海报   |
| 云校长/老师头像 | 1:1，400×400       | testimonial 卡片 |

